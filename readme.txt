== Description ==

Twitter Content Locker plugin is a good solution for website promotion via Twitter. With this plugin you can hide part of your content (for example, images, download links, passwords or other important information) from users. User must share your page on Twitter to view hidden content. Check out <a href="http://www.icprojects.net/twitter-content-locker.html">working demo</a>.

Install the plugin and wrap any "secret" content with shortcodes [twitterlocker] ... [/twitterlocker].

You also can extend standard shortcode by using attribute "message". Example:
[twitterlocker message="Type any message here to replace default message by yours!"] ... [/twitterlocker]

Type shortcode manually or use special button with your WYSIWYG-editor to insert Twitter Content Locker shortcode.

For more details please visit <a href="http://www.icprojects.net/twitter-content-locker.html">Twitter Content Locker</a> plugin page.

== Installation ==

Install and activate the plugin like you do with any other plugins.
